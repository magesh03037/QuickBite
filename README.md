# 🍔 QuickBite - Food Delivery Website

A full stack food delivery website built with Python Django.

## 🌟 Features
- 🏪 Restaurant listings with images
- 🍕 Menu pages for each restaurant
- 🛒 Cart system with quantity tracking
- ✅ Order checkout and confirmation
- 📋 Order history
- 👤 User registration and login
- 🔍 Search functionality
- 🎨 Beautiful responsive design
- ⚙️ Admin panel for management

## 🛠️ Tech Stack
- **Backend:** Python Django 6.0
- **Database:** SQLite
- **Frontend:** HTML, CSS, JavaScript
- **Authentication:** Django Auth

## ⚙️ Installation

### 1. Clone the repository
git clone https://github.com/magesh03037/QuickBite.git

### 2. Install dependencies
pip install -r requirements.txt

### 3. Run migrations
python manage.py migrate

### 4. Create superuser
python manage.py createsuperuser

### 5. Run server
python manage.py runserver

### 6. Open browser
http://127.0.0.1:8000
